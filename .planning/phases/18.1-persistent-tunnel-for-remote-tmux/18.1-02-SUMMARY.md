## User Setup Required

**One-time setup is required before this plan can continue.**

Run these steps in order:

1. **Run the setup guide:**
   ```
   sh /data/home/gsddashboard/scripts/tunnel-setup.sh
   ```
   Follow each printed instruction carefully.

2. **Key steps the guide will walk you through:**
   - `cloudflared login` — opens browser for Cloudflare OAuth (interactive, cannot be automated)
   - `cloudflared tunnel create <name>` — creates the named tunnel
   - `cloudflared tunnel route dns <name> <hostname>` — adds DNS CNAME
   - Add `CLOUDFLARE_TUNNEL_NAME=<name>` to `/data/home/gsddashboard/.env`

3. **Start the tunnel:**
   ```
   sh /data/home/gsddashboard/scripts/tunnel.sh
   ```
   (or install the systemd service for auto-start)

4. **Verify tunnel reachability:**
   ```
   curl -s https://<your-hostname>/api/health
   ```
   Expected: `{"status":"ok",...}`

5. **Then configure Railway:**
   ```
   cd /data/home/gsddashboard && railway variables --set "GSD_DATA_URL=https://<your-hostname>"
   railway up --detach
   ```

After these steps, come back and verify the three plan tasks manually or re-run this plan.
