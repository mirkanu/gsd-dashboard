  <action>
    Create a new component `TasksTab` that accepts `{ projectKey: string }` as its only prop.

    **Structure:**

    ```tsx
    import { useState, useEffect } from "react";
    import { Archive, ArchiveRestore, Plus } from "lucide-react";
    import { api } from "../lib/api";
    import type { GsdTask } from "../lib/types";

    export function TasksTab({ projectKey }: { projectKey: string }) { ... }
    ```

    **State:**
    - `tasks: GsdTask[]` — current visible tasks (open or archived depending on showArchived)
    - `showArchived: boolean` — toggle state, default false
    - `loading: boolean` — true while fetching
    - `title: string` — controlled input for new task title
    - `description: string` — controlled input for new task description (optional)
    - `submitting: boolean` — true while POST is in flight

    **Data loading:**
    - On mount and when `showArchived` changes, call `api.gsd.tasks.list(projectKey, showArchived)` and set `tasks`
    - Cancel in-flight requests on cleanup (use cancelled flag pattern, same as MessageLog in GsdDrawer)

    **Add task form:**
    - Title input: `<input type="text" placeholder="New task title..." required value={title} onChange={...} />`
    - Description input: `<input type="text" placeholder="Description (optional)" value={description} onChange={...} />`
    - Submit button labeled "Add" — disabled when `title.trim() === ""` or `submitting`
    - On submit: call `api.gsd.tasks.create(projectKey, title.trim(), description.trim() || undefined)`, prepend returned task to `tasks`, clear form, set submitting appropriately

    **Open/archived toggle:**
    - A small button or toggle below the form: shows "Show archived" when `showArchived === false`, "Show open" when true
    - Clicking flips `showArchived`

    **Task list:**
    - If `loading`: show `<p className="text-sm text-gray-500 py-4">Loading tasks...</p>`
    - If `tasks.length === 0` and `!loading`: show empty state message appropriate to view ("No open tasks yet." / "No archived tasks.")
    - Otherwise: `tasks.map(task => <TaskRow key={task.id} task={task} onArchive={handleArchive} onUnarchive={handleUnarchive} showArchived={showArchived} />)`

    **TaskRow** (can be inner function or co-located component):
    - Shows `task.title` in `text-sm font-medium text-gray-200`
    - Shows `task.description` below in `text-xs text-gray-500` (only if description is not null/empty)
    - When `!showArchived`: an Archive button (Archive icon, `aria-label="Archive task"`) — on click calls `api.gsd.tasks.update(projectKey, task.id, { archived: 1 })`, removes task from `tasks` state immediately (optimistic)
    - When `showArchived`: an Unarchive button (ArchiveRestore icon, `aria-label="Unarchive task"`) — on click calls `api.gsd.tasks.update(projectKey, task.id, { archived: 0 })`, removes task from `tasks` state immediately (optimistic)

    **Styling conventions:**
    - Form inputs: `w-full bg-surface-3 border border-border rounded px-3 py-1.5 text-sm text-gray-200 placeholder-gray-600 focus:outline-none focus:border-accent`
    - Add button: `px-3 py-1.5 text-xs bg-accent/10 border border-accent/20 text-accent rounded hover:bg-accent/20 disabled:opacity-40`
    - Toggle button: `text-xs text-gray-500 hover:text-gray-300`
    - Task row: `flex items-start justify-between gap-2 py-2 border-b border-border last:border-0`
    - Archive/Unarchive icon button: `text-gray-500 hover:text-gray-300 flex-shrink-0`, icon size `w-3.5 h-3.5`
