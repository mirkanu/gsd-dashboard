PROJECT.md format (from gsddashboard and debates examples):
- Version line: `**Version: v1 — ...` or `**v1.0 is live.**` or `**Live at: https://...`
- More reliable patterns to extract:
  - Version: first occurrence of `\*\*[Vv]ersion:\s*(v[\d.]+)` OR `\*\*(v[\d.]+)` near top of file
  - Live URL: `https?://[^\s)]+` appearing after "live" or "Live at:" near top of file
