  <files>client/src/components/MarkdownViewer.tsx</files>
  <action>
    Create `client/src/components/MarkdownViewer.tsx` — a full-screen overlay that renders a markdown string.

    **Props:**
    ```typescript
    interface MarkdownViewerProps {
      content: string;
      title: string;      // e.g. "State", "Roadmap" — displayed in header
      onClose: () => void;
    }
    ```

    **Behavior:**
    - Renders a `fixed inset-0 z-60 flex flex-col` container (above the drawer's z-50)
    - Background: `bg-gray-950` (solid — not semi-transparent, this is a reading surface)
    - Header: `flex items-center gap-3 px-6 py-4 border-b border-border flex-shrink-0`
      - Back/close button (left): `<ArrowLeft className="w-4 h-4" />` from lucide-react, with `onClick={onClose}`, class `text-gray-400 hover:text-gray-200 transition-colors`
      - Title: `<span className="text-sm font-semibold text-gray-100">{title}</span>`
    - Scrollable body: `flex-1 overflow-y-auto px-6 py-6`
      - Inner content div: `max-w-3xl mx-auto prose prose-sm prose-invert`
      - `<ReactMarkdown remarkPlugins={[remarkGfm]}>{content}</ReactMarkdown>`
    - Escape key listener (same pattern as GsdDrawer):
      ```typescript
      useEffect(() => {
        const handler = (e: KeyboardEvent) => { if (e.key === "Escape") onClose(); };
        window.addEventListener("keydown", handler);
        return () => window.removeEventListener("keydown", handler);
      }, [onClose]);
      ```

    **Imports:**
    ```typescript
    import { useEffect } from "react";
    import ReactMarkdown from "react-markdown";
    import remarkGfm from "remark-gfm";
    import { ArrowLeft } from "lucide-react";
    ```

    **Note on z-index:** The drawer panel uses `z-50`. The overlay backdrop uses `z-40`. MarkdownViewer uses `z-[60]` (Tailwind arbitrary value) to sit above both. If the project's Tailwind config does not extend z-index, use `style={{ zIndex: 60 }}` as a fallback on the outer div.

    Export as a named export: `export function MarkdownViewer(...)`
  </action>
