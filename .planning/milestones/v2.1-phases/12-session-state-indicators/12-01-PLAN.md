  <behavior>
    - archived project (no session name passed as null/undefined) returns "archived"
    - tmux capture-pane output containing "> 1." or "? " or "Press Enter" or "Select an option" returns "waiting"
    - output containing "out of credit" or "insufficient credits" or "rate limit" returns "paused"
    - output with recent non-matching lines returns "working"
    - inactive session (isTmuxSessionActive returns false) returns "paused"
    - any tmux error returns "paused" (fail-safe)
  </behavior>
  <action>
Add `detectSessionState(sessionName)` to `server/gsd/tmux.js`. Export it alongside `isTmuxSessionActive`.

Implementation:
1. If `!sessionName` return `"archived"` (caller passes null for archived projects)
2. If `!isTmuxSessionActive(sessionName)` return `"paused"`
3. Run `execFileSync('tmux', ['capture-pane', '-p', '-l', '50', '-t', sessionName], { encoding: 'utf8' })` in a try/catch (return `"paused"` on error)
4. Classify the captured output (last 50 lines) with these checks in order:
   - `waiting` patterns (any line matches): `/>\s+\d+\./` (numbered list with cursor), `/\?\s/` (question prompt), `/Press Enter/i`, `/Select an option/i`
   - `paused` patterns: `/out of credit/i`, `/insufficient credits/i`, `/rate limit/i`
   - Default: `"working"`
5. Update `module.exports` to include `detectSessionState`

Note: Using `execFileSync` (not `execSync`) for safety — same pattern as `isTmuxSessionActive`.

**A. Add saveConfig() helper** above `loadConfig()`:

```javascript
function saveConfig(config) {
  const configPath = process.env.GSD_PROJECTS_PATH || path.resolve(__dirname, '../../gsd-projects.json');
  fs.writeFileSync(configPath, JSON.stringify(config, null, 2) + '\n', 'utf8');
}
```

**B. Extend GET /api/gsd/projects** (local branch only — proxy branch passes through unchanged):
In the `data` mapping, import `detectSessionState` at top of file alongside `isTmuxSessionActive`, then add:
```javascript
const sessionState = proj.archived
  ? 'archived'
  : detectSessionState(proj.tmux_session ?? null);
return {
  ...readProject(name, root),
  tmuxActive: isTmuxSessionActive(tmux_session),
  sessionState,   // add this field
  contextTokens: row?.context_tokens ?? null,
  sessionUpdatedAt: row?.updated_at ?? null,
};
```
Note: destructure `archived` from each project entry alongside `name`, `root`, `tmux_session`.

**C. Add archive/unarchive endpoints** after the send route, before `module.exports`:
```javascript
// POST /api/gsd/projects/:name/archive
router.post('/projects/:name/archive', (req, res) => {
  if (GSD_DATA_URL) {
    // proxy to upstream
    fetch(`${GSD_DATA_URL}/api/gsd/projects/${encodeURIComponent(req.params.name)}/archive`,
      { method: 'POST', signal: AbortSignal.timeout(10000) })
      .then(r => r.json().then(d => res.status(r.status).json(d)))
      .catch(err => res.status(502).json({ error: 'Failed to reach GSD data source', detail: err.message }));
    return;
  }
  try {
    const config = loadConfig();
    const project = config.projects.find(p => p.name === req.params.name);
    if (!project) return res.status(404).json({ error: 'Project not found' });
    project.archived = true;
    saveConfig(config);
    res.json({ ok: true });
  } catch (err) {
    res.status(500).json({ error: 'Failed to update config', detail: err.message });
  }
});

// POST /api/gsd/projects/:name/unarchive
router.post('/projects/:name/unarchive', (req, res) => {
  if (GSD_DATA_URL) {
    fetch(`${GSD_DATA_URL}/api/gsd/projects/${encodeURIComponent(req.params.name)}/unarchive`,
      { method: 'POST', signal: AbortSignal.timeout(10000) })
      .then(r => r.json().then(d => res.status(r.status).json(d)))
      .catch(err => res.status(502).json({ error: 'Failed to reach GSD data source', detail: err.message }));
    return;
  }
  try {
    const config = loadConfig();
    const project = config.projects.find(p => p.name === req.params.name);
    if (!project) return res.status(404).json({ error: 'Project not found' });
    delete project.archived;
    saveConfig(config);
    res.json({ ok: true });
  } catch (err) {
    res.status(500).json({ error: 'Failed to update config', detail: err.message });
  }
});
```
